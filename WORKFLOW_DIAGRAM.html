Essence Cloud — Kid Tutor Flow

End-to-end flow: child uses the React app, joins a LiveKit room, and the Python agent runs the AI tutor (OpenAI Realtime, optional BitHuman avatar) while lesson pictures stay in sync. 

flowchart TD
    subgraph USER["👤 Child / parent"]
        A([Open app in browser])
        B[Enter name, pick tutor, mode & theme]
        C[Tap Start lesson]
        D[Allow microphone]
        E[See tutor + lesson picture]
        F[Speak answers]
        G([Leave or lesson complete])
    end

    subgraph BROWSER["🖥️ React app — frontend/"]
        H[Build room name
kidtutor-mode-topic-tutor-id]
        I[GET /token]
        J[Join LiveKit room]
        K[Show picture carousel
word index 0, 1, 2…]
        L[Publish child_profile
on data channel]
        M[Listen: lesson_set_index,
pronunciation_result, lesson_complete]
    end

    subgraph TOKEN["🔑 Token server — token_server.py"]
        N[Validate room name]
        O[Return JWT + LiveKit URL]
        P[Serve curriculum images]
    end

    subgraph LIVEKIT["📡 LiveKit cloud room"]
        Q[Real-time audio / video tracks]
        R[Data channel topic: kidtutor]
    end

    subgraph AGENT["🤖 Python agent — agent.py"]
        S[Worker joins room]
        T[Parse room → mode, topic, tutor]
        U[Load word list & AI prompts]
        V[Start OpenAI Realtime voice]
        W{Avatar enabled?}
        X[BitHuman avatar video]
        Y[Greeting: intro + opener question]
        Z[Child replies → start lesson]
        AA{Lesson mode?}
        AB[Vocabulary / Speaking
score pronunciation]
        AC[Quiz
questions about current picture]
        AD[Sync picture index
lesson_set_index / tools]
        AE{More words?}
        AF[Signal lesson_complete]
    end
